# web-art

Archive of interesting web-native art pieces — kept locally for reference.

## Contents

### [`toiletpaper/`](./toiletpaper) — papertoilet.com clone

A local archive of **papertoilet.com** by **Rafael Rozendaal** (2006).

> PAPER TOILET.COM BY RAFAEL ROZENDAAL - 2006 - WWW.NEWRAFAEL.COM
> COLLECTION OF SEBASTIEN DE GANAY - WWW.SEBASTIENDEGANAY.COM
> CODE BY REINIER FEIJEN - WWW.BOXOFCHOCOLATES.NL

One of the classic pieces of web-native net art — a website where you can scroll toilet paper indefinitely. Kept here because the domain could one day go away and the piece is simple enough to host anywhere.

```
toiletpaper/
├── index.html      # Self-contained entry point
└── html/           # Supporting HTML / assets
```

To view: open `toiletpaper/index.html` in a browser.

**Credit** belongs entirely to Rozendaal / Feijen / de Ganay — this repo contains no original work in `toiletpaper/`.
